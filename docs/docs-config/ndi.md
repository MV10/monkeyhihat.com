# [ndi] Section

This section controls network streaming using the [NDI](https://ndi.video/) video protocol. This is primarily for DJ / VJ users and professional production facilities. It is assumed the user has experience with NDI products and software.

The settings for this section are explained in the _[DJ / VJ Notes](https://www.monkeyhihat.com/docs/index.php#/dj-and-vj-notes?id=ndi-support)_ area of the _Appendix_ documentation.
